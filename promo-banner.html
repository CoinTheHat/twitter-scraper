twitter-scraper

Multi-account Twitter/X scraper with rate-limit handling, account rotation & automatic retries. No API key needed.

const scraper = new TwitterScraper();
const tweets = await scraper.search('artificial intelligence', 20);
// → text, author, followers, likes, retweets, views, url
→ Multi-account rotation
→ Rate-limit handling
→ Fallback queries
